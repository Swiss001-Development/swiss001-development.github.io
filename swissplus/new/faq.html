Frequently Asked Questions

Is SwissPlus Free?

To an extent, Yes. 99% of commands are free a the 1% Is for some resource demanding commands.

What's the prefix?

SwissPlus's prefix is + we will soon be adding a moderation section of our bot to allow Mods/Admins to change the prefix as this is sort of a common prefix.

What commands are avaliable for my server?

Members - Command that avaliable for the server will be displayed.
Mods/Admins – If you have the correct permissions for this, you can view ALL commands in the server whether that it's disabled or not.

I'm having an issue from a command i used. What do I do?

You can type +report bug {message} and make sure you include the command, channel, bot perms and the error message.
